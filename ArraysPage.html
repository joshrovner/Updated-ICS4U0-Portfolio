String[] stringArray = {"My", "name", "is", "Josh"}; 
for (int i = 0; i < stringArray.length; i++)
    stringArray[i] += "a";
String temp = stringArray [2];

In this code snippet, line 1 uses shorthand to create a new array with four unique String elements. Then, the second line adds the letter "a" to the end of each element using a for loop. Finally, the last line accesses the element at index 2 (the third element) and assigns it to the "temp" variable.

int[] intArray = new int[4];
for (int i = 0; i < intArray.length; i++)
            intArray[i] = i;

In example 2, we initialize an array of 4 integers. Since the values of each element are not specified, they are set to their default values - 0 for number-related types and null for Strings and objects. Then, we use another for loop, which sets the array at the current index of i to be i, so the element at index 0 is set to 0, the element at index 1 is set to 1, and so on.
Now, we will take a look at arraylists and how to use them efficiently.

ArrayList<String> stringArrayList = new ArrayList<String>(Arrays.asList ("Hello", "there"));
for (String s : stringArrayList)
    s += "z";

In this code, we create an ArrayList of type String and use the Arrays.asList method to insert two Strings. Then, the second line consists of a for-each loop which goes through the whole list and adds a "z" to each element.

ArrayList<Integer> intArrayList = new ArrayList<Integer>();
for (int i = 0; i < 4; i++)
    intArrayList.add (i);
Integer i = intArrayList.remove (2);

The first line in the example declares a new arraylist of type Integer. Remember that primitive data types cannot be used with arraylists. Then, we use a for loop from 0 to 4, adding a new element at the current position of i. Essentially, this just creates a list with the numbers 0 to 4 in that order. Now, since we are using an arraylist, shouldn't a for-each loop instead of a for loop be used? Yes, but in this case, there were no elements to begin with, and as we recall, for-each loops can only modfiy current elements; they cannot create anything new.
The last line assigns an Integer i equal to the result of the remove() arraylist method on the element at index 2. Looking back, the remove method removes the desired element from the list, shifts all the elements behind it to the fill in the empty slot, and returns the value being removed. Thus, this line alters the array to be: 0, 1, 3, 4, and i to be 2, the value that was removed.

Arrays and ArrayLists Tips

Do not mix up array syntax with arraylist methods.
Do not try to access an element before index 0 or after thee last element in the list.
As a rule, only use for-each loops when you want to alter every element in an arraylist.
