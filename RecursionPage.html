Tutorial Continued - Solving a Problem Using Recursion

Let's try to solve a problem using recursion: Using one recursive method, given an integer n, find the facorial of n. No for-loops or built-in methods may be used.

First off, we need to figure out how this can be solved recursively. We know with a factorial, each number is multiplied by one less than itself, until 1 is reached, at which point we stop. Thus, when n equals 1, we've reached the base case. Assuming we name the method "factorial", the code so far would look like this:

public int factorial (int n) 
{
  if (n == 1)
    return 1;

Otherwise, we need to multiply n by n - 1. The final code including this addition would look like this:

public int factorial (int n) 
{
  if (n == 1)
    return 1;
  else
    return factorial (n - 1);
}

Just like this, we can make short, efficient solutions to seemingly complex problems with the power of recursion.

Recursion Tips

Provide a base case to stop infinite recursion.
Trace a method before coding it to understand what it should do.
Alter any necessary paramters with each call to reach the base case.
