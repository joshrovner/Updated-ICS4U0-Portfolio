Recursion Assignment 1 - Using Recursive Solutions to Solve Problems
Reflection: At first, I found the concept of recursion to be quite confusing, but after reading code for recursive functions such as factorial and fibonacci, I started to use it with a strong grasp. To solve the first problem, requiring a method to reverse a parameter's digits, I did not understand how to do so without printing, but then I realized that I could use exponents and the log10 () method to cut off each digit from the back. For the last question, I initially created a hard-coded array and centered my code around it, but then I figured out the correc solution that works with any inputted array.

Objective: To solve a variety of problems utilising recursion with a limit on the number of lines allowed.

The RecursionQ1 class reverses the digits provided in a method call. The RecurionQ2 class takes several strings of input and then prints them back in reverse order. The RecursionQ3 class searches through an array of Comparables and returns true if a certain target element is found, and false otherwise. Finally, the RecursionQ4 class attempts to find an exit out of a matrix built with 1s, representing walls, and 0s, representing empty spaces.
